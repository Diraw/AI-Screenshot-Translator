# 隐私协议

生效日期：2026-03-02

欢迎使用 AI Screenshot Translator（以下简称“本软件”）。在使用本软件前，请仔细阅读本隐私协议。继续使用本软件，即表示你已阅读并理解本协议所述内容。

## 1. 我们收集和处理的信息

本软件在运行过程中，可能会处理以下信息：

- 你主动截取并提交用于识别或翻译的图片内容
- 你在软件中填写的接口配置，例如 API Key、Base URL、模型名称、代理地址等
- 你使用过程中产生的本地历史记录，例如截图缩略图、翻译结果、时间戳、标签等
- 为改进产品体验而记录的匿名使用统计信息（见“第 5 条 Umami 统计说明”）

## 2. 本地数据存储

除非本协议另有说明，本软件会优先将以下信息保存在你的本地设备中：

- 配置文件
- 历史记录
- 截图缓存
- 调试日志（如你主动开启 Debug 模式）

这些数据默认保存在软件配置指定的本地目录中，由你自行控制、修改或删除。

## 3. 第三方 AI 服务说明

本软件本身不直接提供翻译模型能力。你在软件中配置并使用的第三方 AI 服务（例如 OpenAI、Gemini、Claude 或其他兼容接口）会接收你主动提交的图片内容及相关提示词，用于完成识别和翻译。

这意味着：

- 当你发起翻译时，截图内容会被发送到你所配置的第三方服务提供商
- 这些数据如何存储、处理、保留，取决于对应第三方服务商的隐私政策和数据政策
- 你应自行确认并承担所选第三方服务的合规性与使用风险

如果你不希望截图内容被发送到第三方服务，请不要发起翻译请求。

## 4. API Key 与敏感配置

你填写的 API Key 等配置主要用于本地调用你指定的第三方接口。本软件不会主动将这些配置上传到本软件开发者自建服务器，但在以下场景中仍需注意风险：

- 你的配置文件保存在本地设备，若设备被他人访问，相关配置可能被读取
- 若你开启调试功能，日志中可能包含接口请求的调试信息（具体取决于当前版本实现）
- 若你将配置文件或日志主动分享给他人，你应自行承担由此产生的风险

建议你妥善保管设备访问权限，并定期检查或清理本地日志与配置。

## 5. Umami 统计说明

本软件当前集成了 Umami 统计，用于收集有限的匿名使用行为数据，以帮助开发者了解功能使用情况、发现稳定性问题并改进产品体验。

默认情况下：

- Umami 统计开关为启用状态
- 只有在当前版本中成功加载到有效的 Umami 配置时，统计功能才会真正发送匿名事件

你可以在软件中随时关闭该功能：

- 路径为：设置 > 其他 > 高级设置 > 启用 Umami 统计

Umami 统计可能包含以下类型的信息：

- 软件启动、停留、关闭等基础事件
- 部分功能的触发情况（例如打开窗口、使用某项功能等）
- 软件版本、系统环境等非直接身份识别信息

本软件设计目标是不通过 Umami 统计收集以下内容：

- 你的截图图片本身
- 你的翻译正文内容
- 你填写的 API Key
- 你本地保存的历史记录正文

当你关闭“启用 Umami 统计”后，本软件将停止后续常规 Umami 统计上报；如当前统计会话已开始，软件可能发送一次用于结束当前会话的匿名结束事件（如 `session_end`），此后不再继续发送新的统计事件。

但请你理解，任何统计系统都可能存在技术限制或异常情况。如你对匿名统计仍然敏感，请在使用前充分评估风险。

## 6. 我们不主动售卖你的数据

本软件开发者不会以出售个人数据为目的收集或交易你的信息。

## 7. 你的权利与选择

你可以随时通过以下方式控制自己的数据：

- 删除本地历史记录、截图缓存和日志
- 修改或清空 API 配置
- 在设置中关闭 Umami 统计
- 停止使用本软件

- 因本地设备中毒、被入侵、被他人使用而造成的数据泄露
- 你主动上传、分享或导出本地数据所导致的后果
