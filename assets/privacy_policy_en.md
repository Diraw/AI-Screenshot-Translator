# Privacy Policy

Effective Date: 2026-03-02

Welcome to AI Screenshot Translator (hereinafter referred to as "this software"). Please read this privacy policy carefully before using this software. By continuing to use this software, you indicate that you have read and understood the contents of this policy.

## 1. Information We Collect and Process

During operation, this software may process the following information:

- Screenshot content you actively capture and submit for identification or translation
- API configuration you fill in the software, such as API Key, Base URL, model name, proxy address, etc.
- Local history records generated during your use, such as screenshot thumbnails, translation results, timestamps, tags, etc.

## 2. Data Storage Location

- **Screenshot content and translation results**: Uploaded to the translation API service you configure (such as OpenAI, Claude, Gemini, etc.), and the processing is the responsibility of that service provider.
- **API configuration**: Stored locally on your device and will not be uploaded to any server.
- **History records**: Stored locally on your device by default, and you can configure the storage path.

## 3. Data Security

- This software does not actively collect, upload, or share your data.
- All network requests are directly connected to the API service you configure, without going through intermediate servers.
- It is recommended to properly keep your API Key and avoid sharing it with others.

## 4. Third-Party Services

This software supports calling the following third-party translation services:
- OpenAI API
- Google Gemini API
- Anthropic Claude API

When using these services, please refer to their respective privacy policies.

## 5. Disclaimer

- This software is an open-source tool. The author is not responsible for data leakage or losses caused by user misconfiguration.
- Please ensure that the use of translation APIs complies with local laws, regulations, and the terms of use of the respective service providers.

If you have any questions, please submit an issue via GitHub.
